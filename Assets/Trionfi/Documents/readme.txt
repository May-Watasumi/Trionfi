~What is Trionfi?
-------------------------------------------------------------
'Trionfi' is a lightweight language to create conversation scene for games, and Novel Game(Text Adventure Game).
It is provided with full open source(MIT License).
It has most perfectly compatibility with Kirikiri,  major novel game tool in Japan. 


GETTING STARTED
-------------------------------------------------------------
1.Open "Tools/Trionfi/New Project", and you can generato new project if you custom project setting and press 'OK' button.
2. Edit  'project name'/Resources/scenario/start.txt.(It's boot script file)and edit or add UI,other scenario file if you need.
